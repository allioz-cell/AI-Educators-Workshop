Exam tipIf a question mentions viewing angle or color accuracy, the answer is IPS. If it mentions perfect blacks or a pixel that is simply off, the answer is OLED. If it mentions lowest cost or fastest response, it is TN.

Connectivity and configuration

Mobile radios, sharing options and corporate mail settings.

Bluetooth pairing, in order

  1. Turn Bluetooth on for both devices
  2. Put the accessory into discoverable/pairing mode
  3. Select the device from the scan list
  4. Enter or confirm the pairing code
  5. Test that it actually connects and passes audio or input

The final step is testing — CompTIA scores that as part of the procedure.

Sharing a connection

  • Hotspot: the phone becomes a Wi-Fi access point for several devices at once.
  • Tethering: one device shares the cellular link over USB or Bluetooth.
  • Airplane mode: kills every radio at once, then Wi-Fi or Bluetooth can be switched back on individually.
  • NFC: only works within a few centimeters — contactless payment and tap-to-pair.
