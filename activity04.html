Where the damage landsAMD moved from PGA on AM4 to LGA on AM5, so the old shortcut of "Intel is LGA, AMD is PGA" no longer holds. Bent pins on an LGA board are a motherboard replacement; bent pins on a PGA chip are a CPU replacement.

Memory: DDR4 against DDR5

Pin count, voltage and channel design — the three differences the exam tests.

DDR4DDR5
Desktop DIMM pins288288, but the notch sits in a different place
Laptop SODIMM pins260262
Operating voltage1.2 V1.1 V
Typical speeds2133–3200 MT/s4800 MT/s and up
Channel structureOne 64-bit channel per moduleTwo independent 32-bit subchannels per module
Power regulationOn the motherboardPMIC on the module itself
Error checkingECC only on ECC modulesOn-die ECC on every module, plus optional full ECC
No mixingThe generations are not interchangeable. The notch position physically blocks a DDR5 module from seating in a DDR4 slot, and a board supports exactly one of the two.
Populate the right slotsDual channel needs matched modules in the matching colored slots — usually slots 2 and 4. A user who installs two sticks side by side in slots 1 and 2 loses bandwidth without any error message.

M.2 keyingAn M.2 slot is a shape, not a speed. M key gives four PCIe lanes for NVMe, B key gives SATA or two lanes, and a B+M drive fits either. A 2280 drive is 22 mm wide and 80 mm long.

Firmware, boot and platform security

Boot settings, signature checking and the hardware root of trust.

UEFI

Replaced legacy BIOS. Supports drives larger than 2 TB through GPT partitioning, boots faster, and provides a mouse-driven interface. This is where boot order, virtualization support and fan curves live.

Secure Boot

Only allows bootloaders with a trusted signature to run, which blocks boot-level malware. It also blocks unsigned Linux distributions and bootable USB tools, so technicians disable it temporarily and turn it back on afterward.

TPM 2.0

A chip that stores encryption keys in hardware. BitLocker uses it to seal a drive to one specific machine. fTPM is the same function built into the CPU rather than a separate chip.

Classic symptomA desktop that loses the date and time every time it is unplugged, and reports a checksum error at POST, has a dead CMOS battery (CR2032). Settings are stored in CMOS; the battery only keeps them alive when the machine has no power.
